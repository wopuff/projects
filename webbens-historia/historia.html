Internets begynnelse

På 50-talet började kalla kriget mellan USA och Sovjetunionen. USA började inse att de behövde utveckla ett kommunikationssystem som klarade en kärnvapenattack. Vid denna tidpunkt var datorer väldigt stora och det var endast forskare och militären som hade tillgång till dem. Datorerna var tunga och gick inte att flytta på vilket ledde till stor frustration eftersom användare var tvungna att resa långa sträckor för att ta del av datorkraften. Svårigheterna med att använda sådana system fick olika forskare, ingenjörer och organisationer att undersöka möjligheten till ett storskaligt datornät.


Ingen person uppfann internet själv utan när nätverkstekniken först utvecklades tog ett antal forskare och ingenjörer samman sin forskning för att skapa ARPANET (Advanced Research Projects Agency Network). Deras mål var att hjälpa amerikansk militärteknik att ligga före sina fiender och förhindra överraskningar. Bland ARPA:s projekt fanns en uppgift att testa genomförbarheten av ett storskaligt datornät. 1974 skapades TCP/IP som gjorde det enklare för datorer att kommunicera med varandra. TCP/IP står för Transmission Control Protocol/Internet Protocol. Termen används för att beskriva en uppsättning protokoll som styr hur data rör sig genom ett nätverk.


Första användningen av ett datanätverk

1965 lyckades Lawrence Roberts skicka det första meddelandet mellan två datorer. Denna experimentella länk använde en telefonlinje med ett akustiskt kopplat modem som överförde digital data. När tekniken utvecklats ytterligare (TCP/IP) var Leonard Kleinrock den första personen att skicka ett meddelande. Kleinrock försökte skriva "LOGIN" men systemet kraschade efter att bokstäverna "L" och "O" hade dykt upp.

Skillnaden mellan World Wide Web och internet

Internet är nätverksinfrastrukturen som ansluter enheter tillsammans, medan World Wide Web är ett sätt att komma åt information via internets medium. Tim Berners-Lee föreslog först tanken på en "informationswebb" 1989. Den förlitade sig på "hyperlänkar" för att koppla ihop dokument. Skrivet i Hypertext Markup Language (HTML), kan en hyperlänk peka på någon annan HTML -sida eller fil som sitter ovanpå internet.


1990 utvecklade Berners-Lee Hypertext Transfer Protocol (HTTP) och utformade systemet Universal Resource Identifier (URI). HTTP är det språk datorer använder för att kommunicera HTM -dokument över internet, och URI, även känd som en URL, ger en unik adress där sidorna lätt kan hittas. Berners-Lee skapade också en mjukvara som kunde presentera HTML-dokument i ett lättläst format. Han kallade denna webbläsare för World Wide Web.
